# Music Player Web Application

## 📌 Project Overview

This project is a Music Player Web Application developed using HTML, CSS, and JavaScript. It allows users to play, pause, and switch between multiple songs with a modern Glassmorphism user interface.

## 🚀 Features

* Play Music
* Pause Music
* Previous Song
* Next Song
* Auto Play Next Song
* Song Title Display
* Responsive Design
* Glassmorphism UI
* Attractive Music Player Layout

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📂 Project Structure

```text
CodeAlpha_MusicPlayer/
│
├── index.html
├── style.css
├── script.js
├── music.jpg
├── song1.mp3
├── song2.mp3
└── song3.mp3
```

## 🎯 How to Run

1. Download or clone the repository.
2. Add music files:

   * song1.mp3
   * song2.mp3
   * song3.mp3
3. Add an album image named `music.jpg`.
4. Open `index.html` in a web browser.
5. Play and enjoy the music.
