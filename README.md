Music Player
📌 Project Overview

This project is a Music Player Web Application developed using HTML, CSS, and JavaScript. It allows users to play, pause, and switch between multiple songs with a modern Glassmorphism user interface.

🚀 Features
Play Music
Pause Music
Previous Song
Next Song
Auto Play Next Song
Song Title Display
Responsive Design
Glassmorphism UI
Attractive Music Player Layout
🛠️ Technologies Used
HTML5
CSS3
JavaScript

🎯 How to Run
Download or clone the repository.
Add music files:
song1.mp3
song2.mp3
song3.mp3
Add an album image named music.jpg.
Open index.html in a web browser.
Play and enjoy the music.
